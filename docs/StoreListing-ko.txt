PixelQR: QR 코드 생성기 및 스캐너 - 개인정보 보호 우선 & 오프라인 사용 가능

Chrome에서 QR 코드를 생성하고 스캔하는 가장 빠르고 안전한 방법을 경험하세요. 개인정보 보호와 사용 편의성에 중점을 두고 설계된 이 확장 프로그램은 최신 Chrome Manifest V3 기술을 사용하며, 핵심 기능은 완전히 오프라인으로 작동합니다.

✨ 최신 기능

☁️ 로컬 파일/폴더 QR 코드 생성으로 쉬운 공유
• 파일 업로드: 파일 또는 폴더를 Google Drive에 업로드하고 공유 링크의 QR 코드를 자동 생성
• 가시성 제어: 파일 가시성 선택 가능(소유자만 또는 링크가 있는 모든 사람)
• 일괄 처리: 여러 파일 또는 전체 폴더를 동시에 업로드 지원
• 자동 QR 생성: 업로드 후 공유 링크의 QR 코드를 자동 생성하여 빠르게 공유

🎨 내장 로고 지원
• 사전 설정 아이콘: 12가지 인기 플랫폼 내장 로고(Instagram, YouTube, WeChat, Telegram 등)
• 빠른 꾸미기: 내장 로고를 원클릭으로 선택하여 QR 코드를 빠르게 꾸미기
• 사용자 정의 업로드: 사용자 정의 로고 이미지 업로드도 계속 지원
• 투명도 조절: 로고 투명도를 조절하여 QR 코드 스캔 가능성 보장

🔍 URL 스캔 기능
• 원격 스캔: 이미지 URL을 입력하여 QR 코드를 원격으로 스캔 지원
• 로컬 스캔: 로컬 이미지 파일 업로드를 통한 스캔도 계속 지원
• 스마트 인식: QR 코드 콘텐츠 유형을 자동 인식하여 결과 표시

🌙 시스템 다크 모드 자동 적응
• 스마트 전환: 시스템 테마 설정을 자동 감지하여 다크/라이트 모드에 원활하게 적응
• 눈에 편안한 경험: 다크 모드에서 편안한 시각 경험을 제공하여 눈의 피로 감소
• 완벽한 적응: 인터페이스 요소가 자동 조정되어 모든 모드에서 명확하고 읽기 쉽게 보장

🚀 핵심 기능

⚡ 즉시 생성
• 원클릭 생성: 현재 탭의 QR 코드를 즉시 생성
• 스마트 컨텍스트 메뉴: 텍스트, 링크 또는 이미지를 마우스 오른쪽 버튼으로 클릭하여 즉시 QR 코드로 변환
• 맞춤 콘텐츠: 자유 텍스트, URL, Wi-Fi 자격 증명, 이메일, vCard 등을 위한 코드 생성
• 키보드 단축키: Ctrl+Shift+Q로 현재 페이지의 QR 코드를 빠르게 생성, Ctrl+Shift+T로 선택한 텍스트용

📷 고급 스캔
• 로컬 파일 업로드: 로컬 이미지에서 QR 코드를 해독
• URL 스캔: 이미지 URL을 통해 QR 코드를 원격으로 스캔
• 스마트 인식: QR 코드 콘텐츠 유형(URL, 텍스트 등)을 자동 인식
• 결과 작업: 스캔 결과 복사 또는 링크 직접 열기 지원

🎨 전문적인 맞춤 설정
• 완벽한 디자인 제어: 브랜드 스타일에 맞게 색상, 크기 및 여백 조정
• 로고 통합: 내장 로고 또는 사용자 정의 로고를 QR 코드 중앙에 업로드 지원
• 투명도 제어: 로고 투명도(0-100%)를 조절하여 QR 코드 스캔 가능성 보장
• 고품질 내보내기: PNG 형식(최대 1024px)으로 다운로드 지원

📚 기록
• 생성 기록: 생성된 모든 QR 코드를 자동 기록하여 빠르게 액세스
• 스캔 기록: 스캔된 모든 QR 코드 콘텐츠 기록
• 빠른 작업: 기록에서 QR 코드를 빠르게 재생성하거나 복사

🔒 개인정보 보호 및 보안
• 오프라인 우선: 핵심 QR 코드 생성 및 스캔 기능은 완전히 로컬에서 수행되며 네트워크 연결 불필요
• 데이터 개인정보: Google Drive 기능은 사용자가 적극적으로 사용할 때만 Google 서비스에 연결
• 추적 없음: 사용자 데이터, 인터넷 사용 기록 또는 분석 정보를 수집하지 않음
• 최소 권한: 안전한 Manifest V3 아키텍처를 기반으로 구축되었으며, 꼭 필요한 권한만 요청
• 안전하고 깨끗함: 광고, 숨겨진 스크립트, 불필요한 백그라운드 프로세스 없음

💡 왜 우리를 선택해야 할까요?
• 완전 무료: 결제 장벽이나 가입 불필요
• 사용자 친화적: 다크 모드를 완벽하게 지원하는 현대적이고 깨끗한 인터페이스 디자인
• 생산성 향상: 빠른 액세스를 위한 키보드 단축키, 빠른 생성을 위한 우클릭 메뉴 지원
• 글로벌 지원: 12개 언어로 완벽하게 현지화
• 멀티 브라우저 지원: Chrome, Firefox, Edge 브라우저 지원

모바일로 링크를 빠르게 공유하거나, 텍스트 정보를 저장하거나, 인쇄용 전문 QR 코드를 만들거나, Google Drive를 통해 파일을 공유하는 등 모든 요구사항을 PixelQR이 충족합니다. Chrome에서 가장 신뢰할 수 있는 QR 코드 경험을 위해 지금 설치하세요!
