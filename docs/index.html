üöÄ Quick Start

Installation

  • Install from Chrome Web Store, Firefox Add-ons, or Edge Add-ons
  • Click the extension icon in your browser toolbar
  • QR code for the current page will be generated automatically

Keyboard Shortcuts

  • Ctrl+Shift+Q (Windows/Linux) or Cmd+Shift+Q (Mac): Generate QR code for current page
  • Ctrl+Shift+T (Windows/Linux) or Cmd+Shift+T (Mac): Generate QR code for selected text

Usage Tips

  • Right-click on any text, link, or image to generate QR code
  • Customize QR code appearance in the Edit modal
  • Upload files to Google Drive and generate sharing QR codes
  • View history to quickly access previous QR codes
