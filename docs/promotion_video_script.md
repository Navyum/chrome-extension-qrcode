# Pixel QR Code 产品宣传片分镜脚本 (数字人版)

## 🎬 视频概述

* **产品名称**: Pixel QR Code
* **制作方式**: 数字人介绍 + 功能演示截图
* **视觉流程**: 每个分镜先展示数字人，然后切换到功能图片，数字人旁白同步介绍
* **核心宗旨**: 极简分享，跨屏无界，个性定义

## 📝 制作说明

### 视频结构

每个分镜包含两部分：

1. **数字人介绍** (2-3秒): 数字人出镜，执行动作并开始旁白
2. **功能演示** (3-4秒): 切换到功能图片，数字人旁白继续，可叠加功能图片的动画效果

### 动作节奏

* 每个动作持续时间约 0.5-1 秒
* 动作之间自然过渡，保持流畅
* 表情和手势要与旁白内容同步

### 旁白要求

* 语速适中，清晰自然
* 重点词汇可适当强调
* 与画面切换节奏匹配
